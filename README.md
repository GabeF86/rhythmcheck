# RhythmCheck — AI EKG Interpreter

A Progressive Web App (PWA) for anesthesiologists that uses AI to identify EKG rhythms and provide ACLS treatment algorithms with drug doses and OR-specific considerations.

## Deployment to Vercel

1. Push this folder to a GitHub repository
2. Connect the repo to Vercel
3. Deploy — no build step needed

## Adding to iPhone Home Screen

1. Open the deployed URL in Safari on your iPhone
2. Tap the Share button (square with arrow)
3. Tap "Add to Home Screen"
4. The app will now appear as an icon on your home screen
